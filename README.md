# Wok_vue
este desarrollo es creado para wok esta creado con vue se buldea, esta bastante mejor en en frontend y maneja una plantilla

```shell
cd frontend
npm run build
#se buldea en la raiz de la api conc arpeta dist
```

```shell
npm run start
#se creara un servidor statico con la pagina web
```

#se usa pm2
dentro de este codigo se usa pm2 para lograr montar el servidor de node, se usa siempre la ulta version de eslint, webpack se hacer en el vue config y siempre manterner node actualizado
